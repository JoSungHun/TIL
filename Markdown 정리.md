## Heading

#을 사용하여 Heading. (#의 개수에 따라)

Heading은 h1 ~ h6까지 가능함. Heading을 글자크기 조절용으로 사용하지 말것.

# H1

## H2

### H3

###### H6



## Bold, Italic

Bold(굵게)

**을 사용.

굵게  -> **굵게**

Italic(기울기)

*을 사용.

기울기 -> *기울기*

## List

숫자. (1.) 사용하여 순서리스트 사용

1. 순서리스트
2. 순서리스트

스코어 (-) 사용하여 비순서 리스트 사용

- 리스트
- 리스트

## Code

백틱(``) 을 사용하여 코드 표현

`Code`

백틱 3개를 사용하여 (```) 코드블록 가능

```python
Code block
if i:
  print(1)

```
